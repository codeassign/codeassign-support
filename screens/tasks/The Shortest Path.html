The Shortest Path

  • Task
  • Hints
  • Solutions
You are given a list of cities. Each direct connection between two cities has its transportation cost (an integer bigger than $0$). The goal is to find the paths of minimum cost between pairs of cities. Assume that the cost of each path (which is the sum of costs of all direct connections belongning to this path) is at most $200000$. The name of a city is a string containing characters $a, ..., z$ and is at most $10$ characters long. ### Input $s$ : the number of tests $\leq 10$ $n$ : the number of cities $\leq 10000]$ _NAME_ : city name $p$ : the number of neighbours of city _NAME_ $nr$ $cost$: * $nr$ : index of a city connected to _NAME_ (the index of the first city is $1$) * $cost$ : the transportation cost $r$ : the number of paths to find $\leq 100$ _NAME1_ _NAME2_ : _NAME1_ is the source, _NAME2_ is the destination _empty line separating the tests_ ### Output $cost$ : the minimum transportation cost from city _NAME1_ to city _NAME2_ (one per line)
Input #1
``` 1 4 gdansk 2 2 1 3 3 bydgoszcz 3 1 1 3 1 4 4 torun 3 1 3 2 1 4 1 warszawa 2 2 4 3 1 2 gdansk warszawa bydgoszcz warszawa ```
